### Description

The netcode library is a C++11 network coding library, with C interfaces.

### Requirements

- gcc >= 4.7 or clang >= 3.3
- gf-complete (https://github.com/ceph/gf-complete)
- cmake >= 2.8.9

### Building and installing

```
$ mkdir build && cd build
$ cmake ..
$ make && make install
```
